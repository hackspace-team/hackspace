Our Projects

TriV Tracker

We partnered with TriValley CoderDojo, located in Dublin, California, to bring develop an achievement tracking system for the students to use as tool to challenge their understanding of the concepts being learned. We developed this product using front-end tools such as HTML, CSS, and JavaScript. We also used Django, Python, and SQLite for the database and backend of the website.

Pac-Man PVP

We created a rendition of the classic Pac-Man, but modified it to make it a player-vs-player game that two people could enjoy playing together. It excludes the classic ghosts, but instead races the opponents to try and collect more dots that the other! We developed this game using Python and PyGame.
